# Asset Licenses

Current build:

- Procedural SVG props generated for this project.
- Canvas vector character and enemy drawings generated in code.
- Procedural Canvas tile textures generated in code.
- DOM/CSS UI created for this project.
- Existing title key art is project-owned for this deployment.

Rules:

- Do not use unlicensed commercial sprites.
- Do not scrape images from the web.
- Free assets may only be introduced if their license permits web game deployment and style matches the art bible.
- Record every external source, license, author, and URL before committing it.
