# Game Design Document

Title: 盗墓往事 / Tales Beneath the Tomb

Genre: Chinese horror tomb-robbing ARPG for desktop and mobile browsers.

Core loop:

1. Explore a dark place.
2. Read objects and environmental traces.
3. Trigger horror events.
4. Fight or avoid supernatural threats.
5. Collect clues, tapes, equipment, and artifacts.
6. Return to the old house, save, repair, prepare, and descend again.

Current vertical slice promise:

The player controls Chen Guichuan, opens the box on the last bus, reaches the old Chen house in the rain, plays the father tape, triggers the paper effigy event, obtains the rusty key, opens the cellar, and discovers evidence on the artisan name wall.

Design pillars:

- Horror is physical: props, light, sound, and space must carry story.
- Combat has anticipation: enemy windup and silhouettes must be readable.
- Investigation changes objectives: clues are not only lore, they direct the next action.
- Mobile is first-class: controls must be labeled and usable in landscape.
