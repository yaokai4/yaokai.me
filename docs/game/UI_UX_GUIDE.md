# UI / UX Guide

UI is DOM-based in `game/js/ui.js` and styled by `game/css/style.css`.

Core surfaces:

- Title.
- HUD.
- Subtitle.
- Interaction prompt.
- Inventory.
- Journal.
- Quest log.
- Clue board.
- Bench.
- Shop.
- Settings.
- Manual save slots.

Mobile rules:

- Landscape is preferred.
- Touch controls must have glyph plus label.
- Do not hide critical actions behind unlabeled icons.
- Buttons must remain inside the viewport at common landscape sizes.

Accessibility options:

- Subtitle scale.
- High contrast.
- Reduce flash.
- Color assist.
- Horror reduction setting.
