# Game Art Bible

This file mirrors `ART_BIBLE.md` for the prompt requirement.

Authoritative art rules:

- Do not use blurry placeholder art for interactive gameplay props.
- Prefer SVG/Canvas vector assets for runtime readability.
- Do not scrape copyrighted images from the web.
- Do not mix unrelated free asset styles.
- All new assets need stable IDs and license notes.

See `ART_BIBLE.md` and `ASSET_PIPELINE.md` for the implementation details.
